# Antes de inicar
Instalar:
```sh
sudo apt update && sudo apt upgrade
sudo apt install -y python3 pip
sudo apt install -y build-essential libssl-dev libffi-dev python3-dev
```
# Game Project

Para correr el juego debes segur las siguientes instrucciones:
```sh
cd game
python3 main.py
```

# Comandos generales
